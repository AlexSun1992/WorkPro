# Инструкция по подключению локального файла "js для vue форм"

Для подключения локального файла **"js для vue форм"** выполните следующие шаги:

1. **Создайте папку для карточки**:

  - Перейдите в директорию `/components/EventHandler/`.
  - Создайте папку с ID карточки (например, `1089` для карточки Ввод данных ОСАГО).

2. **Создайте файл eventHandler.js**:

  - Внутри только что созданной папки создайте файл `eventHandler.js`.
  - Перенесите все наработки из конфигуратора в этот файл.

3**Очистите конфигуратор**:
  - В конфигураторе удалите весь код.
  - Оставьте только закомментированное сообщение:
    ```js
    // Код перенесён в проект ЛК2, изменения в этом разделе не повлияют на работу карточки
    ```

После выполнения этих шагов при подключении локального файла в **консоли разработчика** будет выведено сообщение:
`localscript: номер карточки`

### Теперь ваш локальный файл подключен и готов к работе!
